Sistema de Escritorios

Esse é um sistema de escritorios completo, usando PHP como linguagem principal.

Para rodar o sistema, você precisa de um servidor local, como o XAMPP, WAMP, LAMP, MAMP, etc.

Instalação

Para instalar o sistema, você precisa clonar o repositório em sua máquina local.

Para clonar o repositório, você precisa ter o Git instalado em sua máquina.

Para clonar o repositório, você precisa rodar o seguinte comando no terminal:

git clone https://github.com/JA-Solutions-Engine/office_system.git office_system

Depois de clonar o repositório, você precisa rodar o servidor local e acessar o sistema pelo navegador.

Para acessar o sistema, você precisa acessar a pasta do sistema pelo navegador, por exemplo:

http://localhost/office_system

Depois de acessar o sistema, você precisa configurar o banco de dados.

Para configurar o banco de dados, você precisa acessar o arquivo config.php na pasta config.

No arquivo config.php, você precisa configurar as constantes de conexão com o banco de dados.

Depois de configurar o banco de dados, você precisa rodar o arquivo database.sql na pasta database no seu banco de dados.

Depois de rodar o arquivo database.sql, você precisa acessar o sistema pelo navegador e fazer login.

Para fazer login, você precisa usar o seguinte usuário:

Usuário: admin

Senha: admin

Depois de fazer login, você pode mudar a senha do usuário admin no sistema.

Depois de mudar a senha do usuário admin, você pode usar o sistema normalmente.
